# PRD — 미래 명함 메이커

## 1. 비전
> "10년 후 너의 명함을 미리 받아본다면?" — 학교 부스에서 받아 가는 가장 가벼운 진로 기념품.

## 2. 사용자
- 초·중·고 학생 (자기 미래 상상)
- 진로 교사 (자유학기제·캠프 활용)

## 3. 사용자 시나리오
1. 이름·신분(초/중/고/대/직장인/기타)·관심 분야 입력
2. (선택) MBTI 모름 → 「🧭 10문항 테스트」 1분 → MBTI·강점 자동 입력
3. (선택) 프로필 사진 업로드
4. Gemini API 키 입력 → 「10년 후 명함 만들기」
5. 앞면 명함 + 클릭 시 3D 회전 → 뒷면 자서전
6. PNG 다운로드 → 인쇄

## 4. 기능 요구사항
- **F1.** 입력 폼 (이름, 신분, MBTI 칩 16종, 관심 분야 다중 선택+직접 입력, 강점, 사진 선택)
- **F2.** MBTI 10문항 빠른 테스트 모달 (4축 균형, 결과로 MBTI/강점 자동 입력)
- **F3.** 프로필 사진 업로드 (4MB 이하, dataURL)
- **F4.** Gemini Text JSON 응답 (한 번에 15개 필드: 회사 / 부서 / 직책 / 한글 슬로건 / 영문 태그라인 / 핵심 역량 3개 / 가상 수상 / 가상 주소 / 이메일 / 웹 / 자서전 제목·본문 / MBTI / 직업 이모지)
- **F5.** 명함 디자인 — 정보 밀도 우선 (빈 공간을 의미 있는 정보로 채움):
  - **상단:** 회사 / 부서·랩 / 직책 / MBTI 배지 / 연도
  - **중앙:** 이름 / 디바이더 / 한글 슬로건 / 영문 태그라인
  - **칩 라인:** 핵심 역량 3개 (그라디언트 칩)
  - **하단 좌:** 이메일 ✉ + 웹 ⊕  
    **하단 중:** 수상 🏆 + 주소 📍  
    **하단 우:** FUTURE 2036 도장
  - 좌측 사이드바: 아바타 (사진 → 이모지 → 이니셜)
- **F6.** 3D 플립 카드 (클릭 시 앞↔뒤 회전, 뒷면=자서전 한 페이지)
- **F7.** PNG 다운로드 (스케일 3, 폰트 로드 보장, 단색 치환, 항상 앞면 캡처)

## 5. 비기능 요구사항
- BYOK — 키 외부 저장 X
- 단일 `index.html` + Vercel
- 텍스트 생성 5~15초
- PNG 출력 시 한글 fake bold 방지

## 6. 디자인 원칙
- 명함 1.7:1 비율
- 좌측 컬러 사이드바(그라디언트 + 도트 패턴)
- 인쇄 친화 베이지 배경(#fcfaf3)
- 뒷면은 인디고/바이올렛 다이어리 분위기

## 7. 성공 지표
- 명함 PNG 다운로드율
- MBTI 테스트 완주율
- "재미있어서 또 만든다" 재방문율 (정성)

## 8. 향후 (Out of scope)
- 사진 업로드 → Gemini Image로 미래 모습 합성
- 양면 명함 PDF 생성
- QR 코드(자서전 링크) 자동 삽입
- 다국어 명함 (영어 비즈니스 명함)
